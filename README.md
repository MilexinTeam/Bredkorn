# Text Editor (TypeScript)

A minimal open‑source **code editor** written in **TypeScript**.  
The project is licensed under the **MIT License**, allowing free use, modification, and distribution.

## Features
- Lightweight architecture  
- Fully implemented in TypeScript  
- Designed as a foundation for building a full code editor  
- Suitable for syntax‑highlighting, plugins, and future developer‑focused features
